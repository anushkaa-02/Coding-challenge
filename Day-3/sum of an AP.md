# Sum of an AP

Problem link: [Click here](https://www.geeksforgeeks.org/problems/sum-of-an-ap1025/1?page=6&difficulty=School&sortBy=submissions)


```cpp
class Solution
{
	public:
		int sum_of_ap(int n,int a, int d)
		{
		    // Code here.
		    return ((n*(2*a+(n-1)*d)))/2;
		}
};
```
